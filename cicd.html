CI/CD Pipeline

This project uses a simple yet powerful CI/CD pipeline built with GitHub Actions to automate deployment to AWS S3.

  • Trigger: Every push to the main branch starts the workflow.
  • GitHub Actions: Runs a workflow file .github/workflows/deploy.yml.
  • Sync: The workflow uses the jakejarvis/s3-sync-action to upload HTML/CSS files to S3.
  • Secrets: AWS credentials and bucket info are securely stored in GitHub Secrets.
  • Automation: Changes are live within seconds, without any manual S3 uploads.

This enables rapid iteration, safe deployment, and fully automated publishing of static websites.
