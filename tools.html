Tools Used

Here are the key tools and technologies used in building and deploying this project:

  • Git & GitHub: For version control and collaboration.
  • GitHub Actions: To automate deployment to S3 on every push.
  • AWS S3: Hosting platform for static website files.
  • HTML & CSS: Front-end structure and styling.
  • MobaXterm Code editor used to build the project locally.
