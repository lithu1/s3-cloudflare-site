Tools Used in This Project

  • Git & GitHub: Version control and collaborative development.
  • GitHub Actions: Automates deployment to AWS S3 on every push.
  • AWS S3: Hosts the static website with public access.
  • HTML & CSS: Used to create a clean, responsive user interface.
  • Cloudflare (optional): CDN, SSL, and DNS for custom domain support.
  • VS Code: Code editor for local development and Git integration.
